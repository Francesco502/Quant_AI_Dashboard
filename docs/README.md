## 🚀 快速开始

详细步骤请参考 [快速启动指南](快速启动指南.md)。

简要步骤：

1. **环境准备**: Python 3.10+, Node.js 18+
2. **安装依赖**:
   ```bash
   pip install -r requirements.txt
   cd web && npm install
   ```
3. **启动服务**:
   - 后端: `python -m uvicorn api.main:app --reload`
   - 前端: `cd web && npm run dev`
   - 后台: `python run_daemon.py`
