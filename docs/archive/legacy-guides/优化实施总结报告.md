## 四、后续使用建议

### 1. 内存监控告警

当内存使用率超过70%时，系统会记录警告日志：

```python
from core.memory_monitor import get_memory_monitor

monitor = get_memory_monitor()
is_healthy, msg = monitor.check_memory()
```

### 2. 强制内存清理

```python
from core.memory_monitor import check_and_cleanup

result = check_and_cleanup()
# 返回清理统计
```

### 3. 交易日判断

```python
from core.trading_calendar import get_trading_calendar, is_trading_day

calendar = get_trading_calendar()
is_today_trading = calendar.is_trading_day()  # 默认A股
is_today_hk = calendar.is_trading_day(market="hk_share")
```

### 4. 模拟交易（推荐）

```python
from core.brokers.paper_broker import PaperBrokerAdapter

broker = PaperBrokerAdapter(user_id=1)
# 进行模拟交易
```
