## 四、部署就绪状态

项目现在已经可以通过以下命令在服务器上部署：

```bash
# 1. 克隆代码
git clone <repo-url> && cd Quant_AI_Dashboard

# 2. 配置环境变量
cp env.example .env
# 编辑 .env 设置 SECRET_KEY 和密码

# 3. 创建必要目录
mkdir -p data logs strategies models

# 4. 构建并启动
docker compose build --no-cache
docker compose up -d

# 5. 访问
# http://your-server-ip (通过 Nginx)
# http://your-server-ip:8685/docs (API 文档)
```
