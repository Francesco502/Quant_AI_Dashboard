# 本地启动前后端服务

> 完整配置说明（.env 项、本地部署步骤）见 [配置与本地部署说明.md](./配置与本地部署说明.md)。

## 后端

```powershell
py -m uvicorn api.main:app --host 0.0.0.0 --port 8685
```

## 前端

在 `web` 目录下先设置 API 地址再启动（二选一）：

- **临时**：`$env:NEXT_PUBLIC_API_URL="http://localhost:8685/api"` 后执行 `npm run dev`
- **推荐**：在 `web/.env.local` 中写 `NEXT_PUBLIC_API_URL=http://localhost:8685/api`，然后直接 `npm run dev`

```powershell
cd web
$env:NEXT_PUBLIC_API_URL="http://localhost:8685/api"
npm run dev
```

## 后端验证

```bash
curl http://localhost:8685/api/health
```
