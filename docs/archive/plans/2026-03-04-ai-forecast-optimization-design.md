## 三、详细设计方案

### 3.1 特征工程增强（10项新特征）

#### 类别一：波动率特征
| 特征名称 | 计算公式 | 说明 |
|----------|----------|------|
| `realized_vol_5` | returns.rolling(5).std() * sqrt(252) | 5日实现波动率 |
| `realized_vol_20` | returns.rolling(20).std() * sqrt(252) | 20日实现波动率 |
| `realized_vol_60` | returns.rolling(60).std() * sqrt(252) | 60日实现波动率 |
| `vol_ratio_5_20` | vol_5 / vol_20 | 短期/长期波动率比值 |

#### 类别二：趋势特征
| 特征名称 | 计算公式 | 说明 |
|----------|----------|------|
| `adx_14` | dx.rolling(14).mean() | 趋势强度指标 |
| `plus_di_14` | (plus_dm.rolling(14).mean() / atr) * 100 | 正方向运动 |
| `minus_di_14` | (minus_dm.rolling(14).mean() / atr) * 100 | 负方向运动 |

#### 类别三：动量特征
| 特征名称 | 计算公式 | 说明 |
|----------|----------|------|
| `momentum_5` | price / price.shift(5) - 1 | 5日动量 |
| `momentum_10` | price / price.shift(10) - 1 | 10日动量 |
| `momentum_20` | price / price.shift(20) - 1 | 20日动量 |
| `streak` | 连续涨跌天数 | 连涨/连跌天数 |

#### 类别四：均值回归特征
| 特征名称 | 计算公式 | 说明 |
|----------|----------|------|
| `zscore_20` | (price - sma_20) / std_20 | 20日Z分数 |
| `zscore_60` | (price - sma_60) / std_60 | 60日Z分数 |
| `bb_position_20` | (price - bb_lower) / (bb_upper - bb_lower) | 布林带位置 |

#### 类别五：价格效率特征
| 特征名称 | 计算公式 | 说明 |
|----------|----------|------|
| `efficiency_ratio_10` | direction / volatility | 10日Kaufman效率比 |
| `efficiency_ratio_20` | direction / volatility | 20日Kaufman效率比 |

### 3.2 模型集成方案

#### 3.2.1 集成模型权重

| 模型 | 基础权重 | 说明 |
|------|----------|------|
| Prophet | 1.0 | 时序趋势预测 |
| XGBoost | 2.0 | 机器学习特征预测 |
| LightGBM | 1.5 | 快速梯度提升 |
| ARIMA | 0.8 | 线性时序建模 |
| **Ensemble** | **2.5** | **多模型自适应集成** |

#### 3.2.2 自适应权重机制

```
动态权重调整逻辑：
1. 计算每个模型在过去30天的预测准确率（MAPE）
2. 权重 = 基础权重 * (1 - MAPE)
3. 归一化权重使其总和为1
4. 使用加权平均得到最终预测
```

### 3.6 信号强度标记规则

#### 3.6.1 信号等级定义

| 信号等级 | 触发条件 | 徽章颜色 | 含义 |
|----------|----------|----------|------|
| `STRONG_BUY` | 涨跌幅 > 15% 且 置信度 > 70% | 红色 | 强烈推荐买入 |
| `BUY` | 涨跌幅 > 10% 且 置信度 > 60% | 橙色 | 推荐买入 |
| `OBSERVE` | 涨跌幅 > 5% 且 置信度 > 50% | 黄色 | 观察仓位 |
| `NEUTRAL` | 涨跌幅 -5% ~ 5% | 灰色 | 中性观望 |
| `SELL` | 涨跌幅 < -5% 且 置信度 > 50% | 蓝色 | 谨慎卖出 |
| `STRONG_SELL` | 涨跌幅 < -10% 且 置信度 > 60% | 紫色 | 强烈推荐卖出 |

#### 3.6.2 信号计算伪代码

```python
def calculate_signal(predicted_return: float, confidence: float) -> Signal:
    if predicted_return > 0.15 and confidence > 0.7:
        return Signal.STRONG_BUY
    elif predicted_return > 0.10 and confidence > 0.6:
        return Signal.BUY
    elif predicted_return > 0.05 and confidence > 0.5:
        return Signal.OBSERVE
    elif predicted_return < -0.05 and confidence > 0.5:
        return Signal.SELL
    elif predicted_return < -0.10 and confidence > 0.6:
        return Signal.STRONG_SELL
    else:
        return Signal.NEUTRAL
```

## 五、性能优化考虑

### 5.1 内存优化策略

| 优化项 | 实施方案 |
|--------|----------|
| 特征缓存 | LRU缓存特征计算结果（maxsize=1000） |
| 模型懒加载 | 按需加载模型，使用完毕后释放 |
| 批量处理 | 分批次处理股票预测，避免内存峰值 |
| 数据压缩 | 使用float32而非float64存储中间结果 |

### 5.2 并行处理

```python
# 使用asyncio并发预测多个股票
async def batch_predict(tickers: List[str]):
    tasks = [predict_single(ticker) for ticker in tickers]
    results = await asyncio.gather(*tasks)
    return results
```

### 6.1 技术风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 内存溢出 | 服务崩溃 | 实现内存监控+自动降级 |
