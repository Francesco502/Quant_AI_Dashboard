## 2. 策略详细设计

### 2.1 动量因子策略 (Momentum Factor)

**原理**: 买入过去表现好的股票，卖出过去表现差的股票

**计算公式**:
```
Momentum = (Price_today - Price_n_months_ago) / Price_n_months_ago
```

**参数**:
- lookback_months: 3, 6, 12 (默认6个月)
- top_percentile: 20 (选择前20%)

**信号评分**:
- 90-100: 动量排名前5%
- 75-89: 动量排名前20%
- 50-74: 动量中等
- 25-49: 动量较弱
- 0-24: 动量排名后20%

**适用场景**: 趋势明显的市场（牛市/熊市）

**风险提示**: 动量崩溃风险，需设置止损

---

### 2.2 MACD策略

**原理**: 通过快线(DIF)与慢线(DEA)的交叉判断趋势

**计算公式**:
```
EMA12 = 12日指数移动平均线
EMA26 = 26日指数移动平均线
DIF = EMA12 - EMA26
DEA = DIF的9日EMA
MACD = 2 * (DIF - DEA)
```

**信号类型**:
1. 金叉买入: DIF上穿DEA且DIF > 0
2. 底背离买入: 价格创新低，DIF未创新低
3. 死叉卖出: DIF下穿DEA
4. 顶背离卖出: 价格创新高，DIF未创新高

**适用场景**: 趋势市场

**优势**: 趋势跟随效果好，信号明确

---

### 2.3 质量因子策略 (Quality Factor)

**原理**: 选择财务质量高的公司

**筛选条件**:
- ROE >= 15% (连续3年)
- 资产负债率 <= 60%
- 经营现金流 > 0
- 盈利波动率 < 行业平均

**评分权重**:
- ROE: 40%
- 负债率: 25%
- 现金流: 20%
- 稳定性: 15%

**适用场景**: 震荡市、价值投资

**数据来源**: 财务报表数据（需接入财务数据源）

---

### 2.4 布林带策略 (Bollinger Bands)

**原理**: 利用价格波动范围判断超买超卖

**计算公式**:
```
中轨 = 20日简单移动平均线
上轨 = 中轨 + 2 * 20日标准差
下轨 = 中轨 - 2 * 20日标准差
带宽 = (上轨 - 下轨) / 中轨
```

**信号类型**:
1. 下轨反弹: 价格触及下轨且RSI < 30
2. 中轨突破: 价格突破中轨向上
3. 带宽收窄突破: 带宽<10%后的放量突破

**适用场景**: 震荡市、趋势转折

---

### 2.5 均值回归策略 (Mean Reversion)

**原理**: 价格偏离均值后会回归

**计算公式**:
```
z_score = (Price - MA20) / Std20
```

**信号**:
- z_score > 2: 卖出信号（价格过高）
- z_score < -2: 买入信号（价格过低）
- |z_score| < 0.5: 平仓信号（回归完成）

**适用场景**: 震荡市场

**风险提示**: 趋势市可能持续偏离

---

### 2.6 资金流向策略

**原理**: 跟随主力资金动向

**指标**:
- 主力资金净流入金额
- 大单买入比例
- 北向资金净流入（港股通）
- 连续净流入天数

**信号评分**:
- 连续3日净流入 + 大单占比>30%: 强烈买入
- 单日净流入>1亿: 买入
- 连续流出: 卖出

**适用场景**: A股市场，短期交易

**数据来源**: Level-2行情数据

---

### 2.7 配对交易策略 (Pairs Trading)

**原理**: 两只相关性高的股票价差会均值回归

**步骤**:
1. 选择同行业相关系数>0.8的股票对
2. 计算历史价差均值和标准差
3. 价差偏离>2标准差时建仓
4. 价差回归时平仓

**适用场景**: 市场中性、震荡市

**风险控制**: 相关性突然下降风险

---

### 2.8 低波动率策略 (Low Volatility)

**原理**: 低波动率股票长期收益更好（低波动异象）

**筛选条件**:
- 过去1年日收益率标准差 < 市场30%分位
- 结合高股息率（>3%）
- 流动性要求：日均成交额>5000万

**适用场景**: 熊市、防御性配置

**优势**: 回撤小，适合保守投资者
