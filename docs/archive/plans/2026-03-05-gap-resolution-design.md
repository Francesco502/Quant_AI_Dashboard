### 2.2 选股结果 → 一键生成调仓计划（复杂算法）

**调仓算法设计**:

```python
# 核心算法：目标权重分配
class RebalancingEngine:
    def generate_plan(
        self,
        current_positions: List[Position],  # 当前持仓
        selected_stocks: List[str],          # 选股结果
        target_cash_ratio: float = 0.2,      # 目标现金比例
        max_position_ratio: float = 0.3,     # 单股最大仓位
        risk_tolerance: str = "moderate"     # 风险偏好
    ) -> RebalancingPlan:
        """
        生成调仓计划
        """
        # 1. 计算目标持仓数量
        total_assets = sum(p.market_value for p in current_positions) + cash
        investable_amount = total_assets * (1 - target_cash_ratio)

        # 2. 等权重分配（可扩展为风险平价、马科维茨优化等）
        target_weight = min(1.0 / len(selected_stocks), max_position_ratio)

        # 3. 计算每只股票的target_value
        for stock in selected_stocks:
            target_value = investable_amount * target_weight
            current_value = sum(p.market_value for p in current_positions if p.ticker == stock)
            diff = target_value - current_value

            if diff > 0:
                # 需要买入
                buy_amount = diff
            else:
                # 需要卖出
                sell_amount = -diff

        # 4. 生成调仓列表
        return RebalancingPlan(
            buys=[...],
            sells=[...],
            expected_cash_after=...,
            expected_positions_after=...
        )
```
