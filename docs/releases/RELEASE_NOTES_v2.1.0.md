# Release Notes `v2.1.0`

## Highlights

- Added a personal asset ledger under the portfolio area.
- Added current holding snapshot tracking, valuation, and PnL summary.
- Added structured DCA rules with trading-day adjustment and auto reconciliation.
- Split research asset pool management from personal holdings management.

## Core Changes

- Backend: new user asset ledger tables and `/api/user/assets*` endpoints.
- Frontend: new personal assets panel with editable holdings and DCA settings.
- Version line bumped from `2.0.1` to `2.1.0`.
