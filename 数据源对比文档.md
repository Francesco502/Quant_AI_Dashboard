# 开源金融数据获取方案对比

除了 yfinance，还有很多优秀的开源数据源可以选择。以下是详细的对比和推荐。

## 📊 数据源分类

### 一、中国A股市场数据源（推荐用于159755等ETF）

#### 1. **AkShare** ⭐⭐⭐⭐⭐
**最推荐用于A股数据**

- **安装**: `pip install akshare`
- **优点**:
  - ✅ 完全免费，无需注册
  - ✅ 数据全面（股票、ETF、基金、期货、期权等）
  - ✅ 支持实时和历史数据
  - ✅ 更新频繁，维护活跃
  - ✅ 支持深交所ETF（如159755）
- **缺点**:
  - ⚠️ 可能有速率限制（需要控制请求频率）
  - ⚠️ 数据格式需要标准化
- **代码示例**:
```python
import akshare as ak

# 获取ETF历史数据
etf_data = ak.fund_etf_hist_sina(symbol="159755", period="daily")
# 或使用通用接口
stock_data = ak.stock_zh_a_hist(symbol="159755", period="daily", adjust="qfq")
```

#### 2. **Tushare** ⭐⭐⭐⭐
**专业级数据源（需要注册）**

- **安装**: `pip install tushare`
- **优点**:
  - ✅ 数据质量高，官方数据源
  - ✅ API稳定，文档完善
  - ✅ 支持多种数据类型
  - ✅ 有积分系统（免费用户有限制）
- **缺点**:
  - ⚠️ 需要注册获取token
  - ⚠️ 免费版有调用频率限制
  - ⚠️ 部分高级功能需要积分
- **代码示例**:
```python
import tushare as ts

# 设置token（需要注册获取）
ts.set_token('your_token_here')
pro = ts.pro_api()

# 获取ETF数据
df = pro.fund_daily(ts_code='159755.SZ', start_date='20230101', end_date='20231231')
```

#### 3. **eastmoney (东方财富)** ⭐⭐⭐⭐
**通过akshare间接使用**

- AkShare 集成了东方财富的数据接口
- 免费且数据全面
- 适合获取ETF、基金数据

#### 4. **MooTdx (通达信)** ⭐⭐⭐
**本地数据源**

- **安装**: `pip install mootdx`
- **优点**:
  - ✅ 可以连接通达信服务器
  - ✅ 支持实时行情
- **缺点**:
  - ⚠️ 需要网络连接通达信服务器
  - ⚠️ 文档相对较少

## 🎯 针对你的项目（159755 ETF）的推荐

### 最佳方案：**AkShare**

**原因**:
1. ✅ 完全免费，无需注册
2. ✅ 完美支持深交所ETF（159755）
3. ✅ 数据更新及时
4. ✅ 安装简单，使用方便

**实现示例**:
```python
import akshare as ak
import pandas as pd

def get_etf_data_akshare(symbol: str, period: str = "daily") -> pd.DataFrame:
    """
    使用AkShare获取ETF数据
    
    参数:
        symbol: ETF代码，如 "159755"
        period: 周期，"daily" 或 "weekly"
    """
    try:
        # 方法1: 使用ETF专用接口
        data = ak.fund_etf_hist_sina(symbol=symbol, period=period)
        
        # 方法2: 如果方法1失败，使用通用股票接口
        if data.empty:
            data = ak.stock_zh_a_hist(
                symbol=symbol, 
                period=period, 
                adjust="qfq"  # 前复权
            )
        
        # 标准化列名
        if not data.empty:
            data.columns = ['日期', '开盘', '收盘', '最高', '最低', '成交量', '成交额', '振幅', '涨跌幅', '涨跌额', '换手率']
            data['日期'] = pd.to_datetime(data['日期'])
            data = data.set_index('日期')
            data = data[['收盘']].rename(columns={'收盘': 'Adj Close'})
        
        return data
    except Exception as e:
        print(f"AkShare获取数据失败: {e}")
        return pd.DataFrame()
```

## 🔧 在项目中集成多数据源

### 方案1: 添加AkShare支持（推荐）

在 `Quant_AI_Dashboard/app.py` 中添加AkShare数据源：

```python
# 在文件顶部添加
try:
    import akshare as ak
    AKSHARE_AVAILABLE = True
except ImportError:
    AKSHARE_AVAILABLE = False
    ak = None

def load_price_data_akshare(tickers: List[str], days: int) -> pd.DataFrame:
    """使用AkShare获取A股/ETF数据"""
    if not AKSHARE_AVAILABLE:
        raise ImportError("AkShare未安装，请运行: pip install akshare")
    
    data_dict = {}
    for ticker in tickers:
        # 移除.SZ/.SS后缀（AkShare不需要）
        clean_ticker = ticker.replace('.SZ', '').replace('.SS', '')
        
        try:
            # 尝试ETF接口
            df = ak.fund_etf_hist_sina(symbol=clean_ticker, period="daily")
            if df.empty:
                # 如果ETF接口失败，尝试股票接口
                df = ak.stock_zh_a_hist(symbol=clean_ticker, period="daily", adjust="qfq")
            
            if not df.empty:
                # 标准化数据格式
                df.columns = ['日期', '开盘', '收盘', '最高', '最低', '成交量', '成交额', '振幅', '涨跌幅', '涨跌额', '换手率']
                df['日期'] = pd.to_datetime(df['日期'])
                df = df.set_index('日期')
                # 只保留最近days天的数据
                df = df.tail(days)
                data_dict[ticker] = df['收盘']
        except Exception as e:
            print(f"获取 {ticker} 数据失败: {e}")
            continue
    
    if not data_dict:
        raise ValueError("无法获取任何数据")
    
    return pd.DataFrame(data_dict)
```

### 方案2: 多数据源Fallback机制

```python
def load_price_data_with_fallback(tickers: List[str], days: int, use_mock: bool = False):
    """多数据源fallback机制"""
    if use_mock:
        return generate_mock_data(tickers, days)
    
    # 尝试顺序：AkShare -> yfinance -> 模拟数据
    sources = [
        ("AkShare", lambda: load_price_data_akshare(tickers, days)),
        ("yfinance", lambda: load_price_data(tickers, days, use_mock=False)),
    ]
    
    for source_name, source_func in sources:
        try:
            data = source_func()
            if not data.empty:
                return data
        except Exception as e:
            print(f"{source_name} 失败: {e}")
            continue
    
    # 所有数据源都失败，返回模拟数据
    print("所有数据源失败，使用模拟数据")
    return generate_mock_data(tickers, days)
```
